# Next.js Community App

최신 Next.js 기반의 간단한 커뮤니티 웹사이트 프로젝트입니다.

## 🔍 프로젝트 개요

본 프로젝트는 Next.js 최신 버전의 기본적인 기능을 활용하여 커뮤니티 웹사이트를 구성합니다.

## 🚀 기술 스택

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS

## 🗂️ 프로젝트 구조

│ ├── app

│ │ ├── page.tsx

│ │ └── [id]

│ └── components
│ └── Navbar.tsx
└── README.md

## ⚙️ 설치 및 실행

```bash
# 프로젝트 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드 및 프로덕션 실행
npm run build
npm run start
```
